프롬프트 이용

npm install -g sass : sass컴파일러 설치
sass --version : sass 컴파일러 버전 확인

sass style/main.scss:style/main.css : 컴파일

sass 변환할 scss 파일명:변환될 css 파일명 : 컴파일
sass --watch 변환할 scss 파일명:변환될 css 파일명 : 실시간 컴파일 (라이브모드를 끌 때 ctrl + c)
sass --style compressed 변환할 scss 파일명:변환될 css 파일명 : 압축형으로

sass --watch --style compressed 변환할 scss 파일명:변환될 css 파일명 : 실시간 압축형으로
